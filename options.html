How to use: Z-Readme Generator

Thank you warmly for downloading the Z-Readme Generator extension! We're excited for you to streamline your project documentation.

Getting Started

  1. To begin, you'll need a Gemini API key. You can easily obtain one from: Google AI Studio.
  2. Once you have your API key, paste it into the input field below and click the "Save" button.

You are good to go! Your settings have been saved. Thank you for using Z-Readme Generator.

How Z-Readme Works

  1. Navigate to your project graph on the Zone01 intra: https://learn.zone01oujda.ma/intra/oujda/profile. From there, select any project you wish to summarize.
  2. Next, choose your desired summary type from the dropdown menu within the extension:
    • Quick Snapshot: Provides a brief overview of your project.
    • Full Breakdown: Generates detailed information about your project.
    • Full Haha: Creates a complete README file (recommended for comprehensive documentation).
  3. Finally, click the "Create README" button to generate the summary based on your selection.
  4. The generated README will appear in the "Output" section of the extension. You can then easily copy the entire README content by clicking the "Copy" button.
